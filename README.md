# 🩺 Ampara - Primeros Auxilios

Aplicación móvil desarrollada en Android que proporciona información básica y práctica sobre **primeros auxilios**, pensada para actuar rápidamente en situaciones de emergencia.

---

## 🚀 Características

* 📚 Información organizada por secciones:

  * Quemaduras
  * Heridas
  * Fracturas
  * RCP (Reanimación Cardiopulmonar)
  * Envenenamiento
  * Emergencias
  * Botiquín
  * Glosario

* 📱 Interfaz simple e intuitiva

* ⚡ Acceso rápido a contenido esencial

* 📖 Funciona sin conexión (offline)

---

## 🛠️ Tecnologías usadas

* Kotlin
* Android Studio
* HTML, CSS y JavaScript (para contenido interno)

## 📦 Instalación

1. Clona el repositorio:

```
git clone https://github.com/nauthix/Ampara-Android-Studio_Primeros_Auxilios.git
```

2. Abre el proyecto en Android Studio

3. Ejecuta la app en un emulador o dispositivo físico
